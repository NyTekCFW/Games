###Pong
A classic Pong game developed in C, utilizing the OpenMLX.

##Command
p : Will Open / Close debug menu (ImGui), perfect to see in real time value and enable some cool features.

Player 1 :
    e = UP
    d = Down
Player 2 :
     o = UP
     l = Down
